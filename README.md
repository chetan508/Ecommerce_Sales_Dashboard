# Ecommerce_Sales_Dashboard
Built an interactive Power BI Retail Sales Dashboard to analyze revenue, profit, customer contribution, and order behavior. Implemented business-accurate KPIs including AOV using DAX, monthly profit trend analysis, and category-wise performance insights using a sample retail dataset.

# Retail Sales Dashboard (Power BI)

## Overview
This project analyzes sales performance of a single retail store using Power BI. 
The dashboard provides insights into revenue, profit, order behavior, customer contribution, and payment trends.

## Key KPIs
- Total Sales
- Total Quantity Sold
- Total Profit
- Average Order Value (AOV)

## Features
- Monthly profit trend with profit/loss analysis
- State-wise sales performance
- Category-wise quantity distribution
- Payment mode analysis
- Top customers by revenue
- Sub-category profit comparison

## Tools Used
- Power BI
- DAX
- Excel (data preprocessing)

## Notes
- This is a sample retail dataset used for learning and analysis.
- Dashboard name does not represent real company data.
